Beyond Asphalt

A Cross-Pavement-Family Explainable Machine Learning Approach to International Roughness Index Prediction Using the LTPP Database

Abstract

International Roughness Index (IRI) is the primary metric highway agencies use to prioritize pavement maintenance, yet nearly every machine-learning model developed to predict it has been trained on asphalt (flexible) pavements alone, leaving rigid pavements dependent on decades-old mechanistic-empirical regression equations. This study addresses that gap. We build a reproducible pipeline directly against the FHWA Long-Term Pavement Performance (LTPP) Standard Data Release and assemble two matched, feature-rich panels: 1,552 flexible-pavement records across 208 sections in 55 U.S. states and Canadian provinces, and 1,693 rigid-pavement records (JPCP, JRCP, and CRCP) across 179 sections in 46 states/provinces, enriched with Falling Weight Deflectometer joint load-transfer efficiency. An Optuna-tuned gradient-boosted ensemble is benchmarked against a mechanistic-empirical-style regression, a random forest, and an Optuna-tuned artificial neural network, with SHapley Additive exPlanations (SHAP) used to interpret the trained model.

Using a section-grouped train/test split that prevents repeated visits to the same physical pavement from appearing on both sides of a fold, the ensemble explains 41.4% of out-of-sample IRI variance for flexible pavements and 22.0% for pooled rigid pavements in a structural variant built from distress, structure, and climate variables alone, rising to 69.8% and 39.1%, respectively, in a companion operational variant that additionally uses each section’s own recent IRI history — two models addressing two distinct engineering questions, reported separately rather than conflated. Traffic loading and load-transfer efficiency, imputed rather than excluded, are shown by ablation to contribute modestly but non-trivially (approximately 0.02 \(R^2\)) to model accuracy, and pooling JPCP/JRCP and CRCP into a single rigid-pavement model is shown empirically to outperform fitting them separately at the present sample size.

SHAP analysis recovers, directly from data, the same distress mechanisms that decades of mechanistic-empirical research encoded by hand — transverse cracking and patching for flexible pavements, wheel-path faulting and load-transfer efficiency for rigid pavements — while additionally surfacing climate interactions that classical closed-form equations cannot represent. Cases in which a modeling intervention (rigid-pavement hyperparameter tuning, the neural-network baseline, splitting JPCP/JRCP from CRCP) did not improve performance are reported with the same emphasis as those that did, consistent with the view that a model’s practical value rests on the reliability of its reported behavior, not on how favorably its results can be selectively presented. The complete pipeline, from raw data acquisition through the trained model and its SHAP explanations, is released as open-source code.

Keywords: International Roughness Index, LTPP, machine learning, SHAP, pavement engineering, flexible pavement, rigid pavement, explainable AI

## Reproducing this work
The figures and tables in this book are pre-computed by the pipeline in [`/code`](https://github.com/sabernaseralavi-60/2026_International-Roughness-Index-Prediction-Using-the-LTPP/tree/master/code)
and read as static assets in the chapters that follow, rather than re-executed at render time, so that
`quarto render` does not require a live LTPP download or a multi-minute model-training pass. Run
`python code/00_download_and_extract_all.py && python code/02_clean_and_merge.py && python code/03_eda.py
&& python code/04_model.py flexible && python code/04_model.py rigid` (plus `rigid_jpcc` / `rigid_crcp`) from
the project root to regenerate everything from scratch.
